de façon à générer automatiquement des @Getter/@Setter,
lombok est autoconfigué au niveau de maven (pom.xml)
mais doit être associé à ECLIPSE (ou autre IDE).

il faut pour cela lancer:
java -jar lombok...jar et choisir l'IDE (eclipse ou ...)
dans la boite de dialogue.

lombok.jar est par exemple dans .m2/repository/org/...
